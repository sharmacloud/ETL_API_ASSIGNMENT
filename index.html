  • To get data using Base_BBL, simply enter http://127.0.0.1:8080/BuildingInfo/base_bbl/YOUR_BBL_ID as the address.

    For example, to get data for Base_BBL=5078630203, the address would be: http://127.0.0.1:8080/BuildingInfo/base_bbl/5078630203

  • To get the average area for the buildings, simply enter http://127.0.0.1:8080/BuildingInfo/getLatestAreaAvearge/ as the address.
